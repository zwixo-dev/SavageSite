# SavageSite

**SavageSite** is a Node.js app that analyzes websites. It uses packages like:  

- **Axios** – to fetch website HTML  
- **Cheerio** – to parse and access the DOM  
- **Express** – to handle routes  
- **fs** – to read/write `.txt` files  

---

## Routes

### Home Page
**URL:** `http://localhost:3000/`  

The home page features **Vanta.js**, which provides an animated 3D background for a cool effect.

### Analyze Page
**URL:** `http://localhost:3000/analyze`  

This page allows you to analyze any website’s structure. It shows:  

- Website title  
- Meta description  
- Headings (H1–H6)  
- Images  
- Links  
- Fonts  

> The page is kept simple for now and is a work in progress.
